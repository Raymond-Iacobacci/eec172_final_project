CC3200 Brick Breaker Game

A showcase of Internet of Things (IoT) capabilities with AWS integration

Project Introduction

This project implements the classic Brick Breaker game on a CC3200 LaunchPad. The aim was to overcome the microcontroller's memory limitations and enhance user experience by integrating AWS IoT for real-time score tracking and storage.

Hardware and Software Tools

  • CC3200 LaunchPad - Microcontroller board for IoT applications.
  • IR Receiver, OLED Display - For remote control and display.
  • Code Composer Studio (CCS) - Integrated Development Environment.
  • AWS IoT Services - For storing and retrieving game scores in real time.

Gameplay Mechanics

Players control a paddle to break bricks by bouncing a ball without letting it drop. Scores are updated and stored via AWS IoT, enhancing gameplay across multiple devices.

Snapshot of the Brick Breaker game in action on the CC3200 LaunchPad.

Technical Challenges

Integrating physical hardware with cloud services presented unique challenges, including handling the data transmission and parsing JSON for the game's score logic.

Future Enhancements

We plan to expand the game's features, including multiplayer capabilities and improved user interface interactions through mobile apps or web services.
